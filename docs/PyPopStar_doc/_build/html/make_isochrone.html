Isochrone¶

class Isochrone(params)

Create a theoretical cluster isochrone.

Parameters:
  • logAge – The age of the isochrone, in log(years)
  • AKs – The extinction of the isochrone, in magnitudes in the Ks filter
  • distance – The distance to the isochrone, in parsecs
  • metallicity – The metallicity of the isochrone, in [M/H]
  • evo_models – The stellar evolution models used (Evolution Model Object)
  • atm_func – The stellar atmosphere models used (Atmosphere Model Object)
  • red_law – The extinction law to be used (Extinction Law Object)
  • mass_sampling – Sample the raw isochrone at every N points, where N=mass_sampling. Must be an integer value.
  • wave_range – A list of length=2 that defines the min/max wavelength of the stellar spectra.
  • min_mass – Set the minimum mass of the isochone, in M_sun
  • max_mass – Set the maximum mass of the isochone, in M_sun
  • rebin – If true, rebin the resolution of the atmospheres so they are the same as the Castelli+04 atmospheres
